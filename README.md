# ShowMyGame
Платформа объединяет разработчиков, игроков и издателей в едином пространстве. Разработчики имеют возможность демонстрировать свои проекты и получать обратную связь без затрат на публикацию, игроки - мгновенно запускать демо в браузере, издатели и работодатели - быстро и просто находить перспективные команды и проекты с помощью фильтров, а также запускать проекты без необходимости их локальной сборки и запуска.
Разработчики могут загружать на сайт демо-версии или прототипы игр, используя технологии, поддерживающие запуск проектов в браузере, такие как WebGL для Unity и Unreal Engine, а также HTML5 для 2D-игр. Платформа автоматически подготавливает загруженные проекты к работе в веб-среде, что позволяет пользователям запускать их мгновенно, без предварительной установки.
# Функции
* Создание карточки проекта с названием, описанием и скриншотом, реализация базового механизма загрузки файлов (WebGL/HTML5-билды).
* Запуск демо в браузере.
* Регистрация и профиль с именем, описанием и аватаром, привязка опубликованных проектов к профилю.
* Сообщения и приглашения к сотрудничеству.
* Присвоение каждому проекту жанра и тегов при публикации.
